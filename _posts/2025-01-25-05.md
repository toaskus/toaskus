### 2. CMD 환경에서 설치



다음의 내용을 명령 프롬프트에서 설치

```
gem install jekyll
```

```
gem install bundler
```



블로그 폴더에서 다음 내용 입력



```
bundle install
```

```
bundle exec jekyll serve
```



http://127.0.0.1:4000/ 확인
